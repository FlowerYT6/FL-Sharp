Welcome to the FL# Tutorial

FL# (Flower Language Sharp) is a simple programming language that you are creating.

Basic Commands:

  • Display Text: display.text('Hello World')
  • Concatenate Text: display.Text('Hello' + ' World')
  • Display Image from URL: display.img('https://example.com/photo.jpg')
  • Display Local Image: display.img('photo.jpg') (Image must be in the same directory)
